# Changelog

---

## Version 1.0.0
**Released:** 2025

Initial release of the Brightnode Attribute System.

**Included:**
- Nine attributes: Health, Stamina, Oxygen, Hunger, Hydration, Temperature, Encumbrance, Mania, Tiredness
- Decay system with Constant, Accelerating, and Decelerating behaviours
- Recovery system with configurable delay and rate per attribute
- Drift system with zone priority handling
- BuildsUp attribute support (Mania, Tiredness)
- Threshold system with AutoApplyModifiers and cascade support
- Modifier system with Stat and Tick modifier types
- Four stacking rules: Stack, Ignore, Refresh, Replace
- Timed modifier support with auto-expiry
- Full multiplayer support - server authority model with client RPC events
- Three preset DataAssets: Human, Animal, Zombie
- 14 included modifier DataAssets
- Three UI widget templates: Radial, Linear, IconDrain
- Debug widget (WBP_AttributeDebugWidget)
- Full demo level with all feature zones
- Complete public API
