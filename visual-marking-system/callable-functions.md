# Core Callable Functions

These are the only functions required to use the BN Visual Marking System.

## Outlines

**Set Outline On Actor**

Applies or removes an outline on a target actor.

Parameters:

- Target Actor - The actor to outline
- Outline Preset - The DA\_OutlinePreset to use
- Enabled - True to apply the outline, false to remove it

## Symbols

**Spawn Symbol Actor**

Spawns a symbol attached to a target actor.

Parameters:

- Target Actor - The actor to attach the symbol to
- Symbol Preset - The DA\_SymbolPreset to use

**Clear Symbol On Actor**

Removes any symbol currently attached to a target actor.

Parameters:

- Target Actor - The actor to clear the symbol from
