# Brightnode Visual Marking System

**Version 1.0.0 | Unreal Engine 5.5+ | Pure Blueprint**

The Brightnode Visual Marking System gives you a clean, data-driven way to apply outlines and symbols to actors in your world. Everything is driven by preset data assets and callable functions — no custom rendering setup beyond enabling stencil.

## Key Features

- Pure Blueprint — no C++ required
- Outline system — custom depth stencil outlines with configurable presets
- Symbol system — spawn and clear world-space symbols attached to actors
- Data driven — all presets defined in DA_OutlinePreset and DA_SymbolPreset data assets
- Preset library — central registry for all presets
- Helper functions — toggle-safe checks for current outline and symbol state

## Requirements

| Requirement | Detail |
|---|---|
| Engine Version | Unreal Engine 5.5+ |
| Implementation | Pure Blueprint |
| Plugins Required | None beyond built-in |
| Custom Depth Stencil | Must be enabled with Stencil |
