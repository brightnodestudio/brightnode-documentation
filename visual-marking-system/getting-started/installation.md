# Installation & Setup

## Adding to Your Project

1. Purchase and download the Brightnode Visual Marking System from the [Fab store](https://www.fab.com/listings/f0beaa5d-7825-486e-af5a-6114002c74b7)
2. In the Content Browser of the BN Visual Marking System project, right click the folder **BN\_VisualMarkingSystem**
3. Choose **Asset Actions - Migrate**
4. Migrate the folder into your new project's Content directory

## Enable Custom Depth Stencil

`Edit - Project Settings - Rendering`

Search for **Custom Depth-Stencil Pass** and set it to **Enabled with Stencil**.

Restart the editor if prompted.

This is required for outlines to render.

## Add the Visual Marking Component

Open your character Blueprint and add the component **AC\_VisualMarkingSystem**.

Compile and Save.

This component is the only system-level requirement on the character.
