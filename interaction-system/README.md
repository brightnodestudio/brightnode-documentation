# Brightnode Interaction System

**Version 1.0.0 | Unreal Engine 5.5+ | Pure Blueprint**

The Brightnode Interaction System is a multiplayer-ready interaction framework built entirely in Blueprint. Configure interaction types via data assets, add the component to your character, and interact with any actor in the world with full UI prompt support and optional outline highlighting.

## Key Features

- Pure Blueprint — no C++ required
- Multiplayer ready — server authority model
- Data driven — all interaction behaviour defined in DA_Interactable data assets
- Multiple interaction types — Press, Hold, Grab, Physics Grab
- Focus trace and radius scan — flexible detection system
- Outline highlighting — optional outline on focused interactables
- Common UI — input-aware prompt icons for keyboard and gamepad

## Requirements

| Requirement | Detail |
|---|---|
| Engine Version | Unreal Engine 5.5+ |
| Implementation | Pure Blueprint |
| Plugins Required | Common UI (built into UE5) |
| Multiplayer | Full replication supported |
