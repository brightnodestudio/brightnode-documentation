If you need help, want to report a bug, or want to stay up to date with new systems and updates:

The fastest way to get support. Join the Brightnode Studio Discord to ask questions, report bugs, and see what is coming next.

💬 [discord.gg/VSu4A7XxA7](https://discord.gg/VSu4A7XxA7)

- **#bugs-attribute-system**
- **#bugs-perk-system**
- **#bugs-interaction-system**
- **#bugs-visual-marking-system**

When reporting a bug please include:

- Unreal Engine version
- System version if applicable
- Clear steps to reproduce the issue
- Screenshots or short videos where possible
