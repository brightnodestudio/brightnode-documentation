# Installation & Setup

## Adding to Your Project

1. Purchase and download the Brightnode Perk System from the [Fab store](https://www.fab.com/listings/6a0214f9-1c4a-4c40-bb96-bbbe5008c491)
2. Open the Epic Games Launcher and navigate to your Fab Library
3. In the Content Browser of the BN Perk System project, right click the folder **BN\_PerkSystem**
4. Choose **Asset Actions - Migrate**
5. Migrate the folder into your new project's Content directory
6. Delete the Showcase folder if you do not need the example content

## Plugin Requirements

The system requires the **Common UI** plugin.

`Edit - Plugins - Common UI`

Restart the editor when prompted.

## Game Viewport Client

Open:

`Project Settings - Engine - General Settings`

Set:

Game Viewport Client Class - **CommonGameViewportClient**

This is required for the Common UI input system.
