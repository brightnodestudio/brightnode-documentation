# Brightnode Perk System

**Version 1.0.0 | Unreal Engine 5.5+ | Pure Blueprint**

The Brightnode Perk System is a modular, multiplayer-ready perk system inspired by the Zombies-style perk machine format. Drop a perk machine into your level, assign a perk data asset, and your players can purchase and equip perks with full UI feedback and animation support.

## Key Features

- Pure Blueprint — no C++ required
- Multiplayer ready — server authority model
- Data driven — all perks defined in PDA_PerkDefinition data assets
- Modular perk machines — create custom machines via child classes
- Power switch system — optional power mechanic for machines
- Currency interface — plug in your own currency system or use the included showcase component
- Animation support — drink sequence with socket attachment
- UI feedback — included WBP_PerkBar widget

## Requirements

| Requirement | Detail |
|---|---|
| Engine Version | Unreal Engine 5.5+ |
| Implementation | Pure Blueprint |
| Plugins Required | Common UI (built into UE5) |
| Multiplayer | Full replication supported |
